## Overview

This folder contains the datasets used in the **KEMET – Egyptian Museums Management & Analytics Platform**. The data represents museum operations, visitors, bookings, tours, events, guides, and cultural heritage assets across Egyptian museums between 2021 and 2026.

## Dataset Descriptions

### Museum
Contains museum details such as museum name, location, category, capacity, and operating hours.

### Hall
Contains information about exhibition halls, their capacities, and associated museums.

### Artifact
Contains artifact information including category, historical era, and assigned museum hall.

### Visitor
Contains visitor demographic information, nationality, and registration details.

### MuseumBooking
Contains museum ticket booking transactions including booking dates, ticket quantities, and booking status.

### Tour
Contains guided tour information including duration, language, guide assignment, and capacity.

### Guide
Contains guide profiles, ratings, and experience information.

### Language
Contains supported languages available for guided tours.

### GuideLanguage
Maps guides to the languages they can provide tours in.

### TourBooking
Contains tour reservation records including ticket quantities, occupancy rates, and revenue information.

### Event
Contains museum event details including event schedules and capacities.

### EventBooking
Contains event reservation and attendance records.

### Pricing
Contains pricing configurations based on service type, ticket type, ticket class, and nationality category.
