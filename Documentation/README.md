## 🏛️ KEMET Platform

KEMET is an end-to-end Business Intelligence solution that integrates:

* SQL Server
* SSIS
* SSRS
* Power BI
* Node.js
* Data Warehouse Technologies

to support data-driven decision-making for Egyptian museums through reporting, analytics, and AI-powered insights.
